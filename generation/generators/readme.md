# Generators - ❌ УДАЛЕНО

## Текущее состояние

Генераторы как отдельные классы **УДАЛЕНЫ**. Их логика перенесена в классы типов (`app\types\*Type`).

## Историческая справка

Раньше генераторы были отдельными классами и находились в этой директории. Каждый генератор реализовывал `GeneratorInterface` и отвечал за создание значений для своего типа атрибута.

**Почему удалено:**

- Дублирование ответственности между Type и Generator классами
- Лишняя сложность с `GeneratorResolver`
- Type классы теперь сами содержат логику генерации

## Что осталось

- `GeneratorInterface.php` - интерфейс который теперь включает `AttributeTypeInterface`
- Этот README - как историческая справка
