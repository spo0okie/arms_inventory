# Библиотека работы с расписаниями

Эта библиотека предоставляет функции для работы с временными интервалами и расписаниями.

## Основные функции

- `intervalsAdd` - добавление интервала к существующим интервалам
- `intervalsSubtract` - удаление интервала из существующих интервалов
- `intervalsIntersect` - нахождение пересечения интервалов
- `intervalsUnion` - объединение интервалов

## Установка

Библиотека не требует отдельной установки, так как является частью проекта.

## Запуск тестов

Для запуска тестов выполните следующую команду в директории `modules/schedules/compile/lib/js`:

```bash
npm test
```

Тесты используют Jest framework и покрывают основные функции библиотеки.

## Структура проекта

- `demo.js` - основная библиотека с функциями работы с интервалами
- `demo.test.js` - тесты для библиотеки
- `package.json` - конфигурация npm с зависимостями и скриптами
