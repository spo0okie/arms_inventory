# Схема БД модуля Schedules

## Таблица `schedules`

Основная таблица расписаний. Хранит заголовок расписания, период его действия и ссылки на иерархию.

| Поле | Тип | NULL | Описание |
| ---- | --- | ---- | -------- |
| `id` | INT | NOT NULL | Первичный ключ, AUTO_INCREMENT |
| `name` | VARCHAR(255) | NOT NULL | Название расписания |
| `description` | VARCHAR(255) | NULL | Пояснение (отображается в списке) |
| `parent_id` | INT | NULL | FK → `schedules.id` — родительское расписание (наследование) |
| `override_id` | INT | NULL | FK → `schedules.id` — расписание, которое это перекрывает |
| `start_date` | VARCHAR(64) | NULL | Дата начала действия расписания (формат `Y-m-d`) |
| `end_date` | VARCHAR(64) | NULL | Дата окончания действия расписания (формат `Y-m-d`) |
| `history` | TEXT | NULL | Заметки / история изменений (произвольный текст) |

### Индексы таблицы `schedules`

| Индекс  | Поля | Тип         |
|---------|------|-------------|
| PRIMARY | `id` | PRIMARY KEY |

## Таблица `schedules_entries`

Записи расписания — временные интервалы на конкретные дни, дни недели или периоды.

| Поле | Тип | NULL | Описание |
| ---- | --- | ---- | -------- |
| `id` | INT | NOT NULL | Первичный ключ, AUTO_INCREMENT |
| `schedule_id` | INT | NULL | FK → `schedules.id` — расписание-владелец |
| `date` | VARCHAR(64) | NULL | День/дата: `'1'`–`'7'` (пн–вс), `'def'` (по умолчанию), `'YYYY-MM-DD'` (конкретная дата), или дата начала периода |
| `date_end` | VARCHAR(64) | NULL | Дата окончания периода (только для `is_period = 1`) |
| `schedule` | VARCHAR(255) | NULL | Временные интервалы: `'08:00-17:00'`, `'08:00-12:00,13:00-17:00'`, `'-'` (выходной) |
| `is_period` | BOOLEAN | NULL | `1` — запись является периодом (диапазон дат), `0` — запись на день |
| `is_work` | BOOLEAN | NULL | `1` — рабочий период / доступ предоставляется, `0` — нерабочий / доступ отозван |
| `comment` | VARCHAR(255) | NULL | Комментарий к записи |
| `history` | TEXT | NULL | История изменений записи |
| `created_at` | TIMESTAMP | NULL | Время создания записи |

### Индексы таблицы `schedules_entries`

| Индекс | Поля | Тип |
| ------ | ---- | --- |
| PRIMARY | `id` | PRIMARY KEY |
| `idx-schedules_days_org_id` | `schedule_id` | INDEX |
| `idx-schedules_days_date` | `date` | INDEX |
| `idx-schedules_days_end_date` | `date_end` | INDEX |
| `idx-schedules_days_is_period` | `is_period` | INDEX |
| `idx-schedules_days_is_work` | `is_work` | INDEX |

### Форматы поля `date`

- `'1'`..`'7'` - день недели (понедельник=1, воскресенье=7)
- `'def'` - расписание по умолчанию (применяется если нет записи на конкретный день)
- `'YYYY-MM-DD'` - конкретная дата (исключение)

### Форматы поля `schedule`

| Значение | Описание |
| -------- | -------- |
| `'-'` | Нерабочий день (выходной) |
| `'08:00-17:00'` | Один рабочий период |
| `'08:00-12:00,13:00-17:00'` | Несколько периодов (с перерывом) |
| `'22:00-06:00'` | Период, переходящий на следующий день |
| `'08:00-17:00{meta}'` | Период с метаданными (опционально) |

## Связи с другими таблицами

### Внешние ключи, исходящие из модуля

```text
schedules.parent_id   → schedules.id        (самоссылка: иерархия наследования)
schedules.override_id → schedules.id        (самоссылка: перекрытия)
schedules_entries.schedule_id → schedules.id
```

### Внешние ключи, входящие в модуль (из других таблиц)

```text
services.providing_schedule_id → schedules.id   (расписание предоставления сервиса)
services.support_schedule_id   → schedules.id   (расписание поддержки сервиса)
acls.schedules_id              → schedules.id   (расписание доступа)
maintenance_jobs.schedules_id  → schedules.id   (график регламентных работ)
```

### Диаграмма связей

```text
┌─────────────────────────────────────────────────────────────────┐
│                         schedules                               │
│  id ◄──────────────────────────────────────────────────────┐    │
│  parent_id ──────────────────────────────────────────────► id   │
│  override_id ────────────────────────────────────────────► id   │
└─────────────────────────────────────────────────────────────────┘
         ▲                    ▲                    ▲           ▲
         │                    │                    │           │
services.providing_    services.support_    acls.schedules_  maintenance_jobs.

│                      schedules_entries                          │

```
