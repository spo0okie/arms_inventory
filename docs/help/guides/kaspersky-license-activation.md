# Антивирус Касперского: как активировать лицензии

Инструкция по получению файлов ключей по кодам активации и добавлению их в Kaspersky Security Center.

Общий сценарий:

1. Запросить коды активации у вашего поставщика лицензий (обычно их присылают на почту).
2. Зайти на сайт <https://keyfile.kaspersky.com/en-US> и ввести присланный код активации. Файл ключа придёт на указанный вами E-mail.
3. Зайти в Kaspersky Security Center и добавить в него полученные файлы ключей.

> Конкретный порядок получения кодов активации зависит от вашего поставщика лицензий — уточняйте у него.
