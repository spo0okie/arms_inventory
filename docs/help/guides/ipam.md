# IPAM — карта адресного пространства

Карта показывает, как занято адресное пространство: слева — крупные
блоки, вправо — всё более мелкая нарезка, до размера «Детализации».
Каждая ячейка — блок адресов; по ней видно, свободен он, занят целиком
объявленной [сетью](../models/networks.md) или содержит вложенные сети.

## Раскраска

- **Цветная ячейка** — блок принадлежит сегменту: цвет задаёт маркер,
  выбранный у [сегмента](../models/segments.md) сети (на карте
  используется только фон маркера, рамка не показывается);
- **полосы нескольких цветов** — внутри блока сети разных сегментов;
  ширина полосы пропорциональна числу сетей каждого сегмента;
- **штриховка** — внутри блока есть вложенные сети мельче размера
  ячейки; их список показывает подсказка при наведении;
- **белая** — сеть (или доля блока) без сегмента или с сегментом
  без маркера;
- **светло-серая** — блок свободен, сетей внутри нет.

## Куда кликать

- **Свободная ячейка** — открывает форму создания сети с уже
  заполненным адресом/маской этого блока;
- **ячейка со штриховкой** (вложенные сети) — «зум»: карта
  перестраивается на этот блок с более мелкой детализацией;
  у точно совпадающей сети с вложенными для зума служит бейдж «+N»;
- **ячейка с сетью** — переход к карточке сети;
- **кнопка «↑ Вверх»** над картой — возврат на уровень крупнее.

## Параметры

- **Базовый адрес** — с какого адреса начинается карта;
- **Маска** — размер всего отображаемого блока (левая колонка);
- **Детализация** — размер самой мелкой ячейки (правая колонка).
  Разница «Детализация − Маска» задаёт число строк; сети мельче
  детализации отдельных ячеек не получают — они видны как штриховка
  и раскрываются зумом.
