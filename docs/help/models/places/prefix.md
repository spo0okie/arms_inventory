# Помещения → Префикс

Используется для [автоматического именования оборудования](../techs.md#инвентарный-номер). Также наследуется от родительского помещения, поэтому нужно выставлять только в

- Корневом объекте (филиал)
- Если внутри филиала есть место, где используется другой префикс для инвентарников

Желательно выставлять такой же, как и выбран для АРМ в этом филиале, чтобы оборудование именовалось похоже на АРМ.
