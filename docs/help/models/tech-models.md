# Модели оборудования

Любое оборудование, которое мы купили, имеет какое-то наименование модели. Все они заводятся в отдельную таблицу.
Зачем? Почему нельзя вписывать модель от руки в каждый экземпляр ПК/оборудования?
Чтобы не было опечаток и чтобы, внеся модель один раз, сразу описать целую пачку одинаковых системников/принтеров/коммутаторов. А дополняя описание модели, вы актуализируете спецификацию сразу всех её экземпляров.

## Список

Наиболее удобный способ смотреть модели — открыть через меню
**[Оборудование -> Категории оборудования -> Категория](tech-types.md)**
и увидеть список моделей какой-то одной категории (МФУ/ноутбуков/коммутаторов).

Есть и общий список всех моделей всех категорий (меню **Оборудование → Модели оборудования**), но пользоваться им менее удобно — как правило, модель ищут в рамках её категории.

## Добавление

Чтобы добавить оборудование, надо, чтобы в инвентаризации уже была модель добавляемого оборудования. Если её нет — нужно в окне [просмотра категории оборудования](#список) нажать кнопку **«Добавить»**.

> Если вдруг вы добавляете оборудование, категории для которого ещё не существует — сначала надо [создать новую категорию оборудования](tech-types.md).
> Также если в форме добавления модели не удастся выбрать производителя оборудования, то надо сначала создать нового вендора.

Откроется пустая форма редактирования модели оборудования. Категория оборудования в ней будет предзаполнена из окна, из которого мы вызвали добавление модели.

При заполнении формы главное — внести точное [Наименование](tech-models/name.md) модели и, пользуясь подсказкой из [шаблона описания категории](tech-types/comment.md), заполнить [Описание](tech-models/comment.md).
