# Типы лицензий

**Тип лицензии** — это полное/исчерпывающее описание лицензии в плане «как она работает».

Он состоит по сути из комбинации:

- [схемы лицензирования](lic-types.md)
- списка ПО, покрываемого этим типом лицензии

Что это значит? Проще всего объяснить на примере: есть лицензии на Windows 10 Pro, но они бывают разных типов. Например:

Лицензии **Windows 10 Pro OEM** лицензируют:

- Windows 10 Pro
- Windows 8.1 Professional (до тех пор, пока она поддерживается MS)
- Windows 7 Professional (до тех пор, пока она поддерживается MS)

### Источник (OEM)

[Лиц. соглашение](https://www.microsoft.com/en-us/Useterms/OEM/Windows/10/UseTerms_OEM_Windows_10_Russian.htm), пункт 7
*(правда, никак не написано **как** активировать такие установки, но у меня, например, был опыт, когда надо было просто звонить на линию поддержки и диктовать (а позже набирать с телефона) туда код от более высокой версии (Win 7), чтобы получить код на более низкую (Win XP))*

По схеме лицензирования **OEM**: лицензия на ПО закрепляется за физическим ПК, с которым была приобретена, и не может быть установлена на другой ПК.

Лицензии **Windows 10 Pro BOX** лицензируют:

- Windows 10 Pro

### Источник (BOX)

[Лиц. соглашение](https://www.microsoft.com/en-us/Useterms/Retail/Windows/10/UseTerms_Retail_Windows_10_Russian.htm), пункт 7
(условие *«Если вы приобрели устройство у изготовителя или установщика с профессиональной версией Windows, предустановленной на нём»* — на BOX-редакции не выполняется)

По схеме лицензирования **BOX**: лицензия на ПО закрепляется за физическим ПК, но может быть перенесена на другой ПК.

Т.е. лицензии на Windows 10 бывают разные, работают по-разному, и всё несколько сложнее, чем просто «лицензии на Windows 10». Поэтому надо разбираться, что конкретно за лицензии мы покупаем, и закупки лицензий разного типа группировать по типам отдельно.

## Список

Доступен через главное меню **Лицензии => Типы лицензий**.
Из требующих пояснения столбцов:

- Закупок акт/всего — количество активных (не истекших) / общее количество закупок
- Ключей исп/всего — количество распределённых / всего активных ключей

В остальном всё должно быть интуитивно понятно:

- Чтобы перейти к просмотру — достаточно кликнуть на любой элемент в списке в столбце «Описание»
- Чтобы перейти к редактированию — можно кликнуть карандаш

## Редактирование

- Заполняем список лицензируемого ПО (программные продукты берутся из реестра ПО)
- Выбираем [схему лицензирования](lic-types.md)
- Вносим описание типа лицензий
- Если надо распределять свободные лицензии — заполняем списки «АРМы, куда распределять лицензии», «ОС/ВМ, куда распределять лицензии» и/или «Пользователи, на которых распределять лицензии»
- Вносим комментарии, если осталось что-то важное, что не вошло в другие поля
