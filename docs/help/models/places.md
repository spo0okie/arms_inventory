# Помещения

Дерево физических локаций организации (филиал → здание → этаж → кабинет). По помещениям раскладывается оборудование — чтобы было видно, где физически что находится. На основе помещений строится карта рабочих мест.

## Список

Список помещений открывается через меню **Организация → Помещения**: пункт «Помещения» разворачивается в подменю со списком филиалов, а в конце подменю есть пункт **«Список помещений»** — он и открывает полный список.

Откроется дерево помещений, где на корневом уровне находятся филиалы.
В филиалах нет строгой формы, как делать уровни помещений. Если все сотрудники размещаются в пределах одного этажа, то нет смысла расписывать здания и этажи — но и лишние уровни ничего не сломают, отображение останется понятным.

## Добавление

- Чтобы добавить внутри помещения другое (кабинет внутри этажа например) — надо нажать кнопку «+» напротив родительского помещения.
- Чтобы добавить корневое помещение — надо нажать в шапке кнопку «Добавить». В случае, когда ни одно помещение еще не заведено, новое создать можно только так (т.к. нет ни одного родительского, чтобы сделать дочернее).

## Редактирование

Чтобы изменить помещение — жмем напротив него карандаш.
Назначение полей описано в подсказках формы, а про [Префикс](places/prefix.md) есть отдельная страница.
