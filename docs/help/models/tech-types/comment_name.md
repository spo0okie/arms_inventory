# Категории оборудования → Замена комментария

Если договорились, что какому-то оборудованию в поле **«Примечание»** всегда надо писать одно и то же, то можно переименовать это поле, чтобы было видно, что заполнять. У телефонов написано **«Внутренний номер»** — это значит, что когда откроется форма редактирования/заведения оборудования и у оборудования будет выбрана модель оборудования из категории телефонов, поле «Примечание» начнёт называться «Внутренний номер», явно намекая, что там надо написать не абы что.

Если название поля заменили, то параметром «Пояснение поля комментария» можно заменить и его подсказку.
