# Категории оборудования

Оборудование делится на категории — и для группировки, и для раскраски, и для небольшого кастомного функционала, который закрепляется через поле «Может выполнять роли».

## Роли

Роль определяет, как оборудование категории ведёт себя в составе [АРМ](techs.md):

- категория с ролью «Компьютер» квалифицируется как АРМ, к которому можно подцепить любое другое оборудование (автоматизированное рабочее место без принтера может быть, а без компьютера — нет);
- дисплеи, ИБП и телефоны выводятся в строке «комплектации» АРМ;
- номер телефона будет присвоен пользователю АРМ как рабочий внутренний;
- если дисплей есть в составе АРМ, то обнаруженные скриптами мониторы будут скрываться.

## Редактирование

Помимо ролей, категория задаёт:

- [Префикс инв. номера](tech-types/prefix.md) — свой префикс в инвентарных номерах оборудования этой категории;
- [Шаблон описания моделей](tech-types/comment.md) — подсказку для однотипного описания характеристик моделей оборудования;
- [Замену комментария](tech-types/comment_name.md) — переименование поля «примечание» (и его подсказки) у оборудования этой категории.

Категории оборудования выводятся в главном меню (**Оборудование → Категории оборудования**) отдельными пунктами; ненужные в меню категории скрываются галочкой «Не отображать в меню».
