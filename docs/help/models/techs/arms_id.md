# Оборудование → Рабочее место

Можно выбрать АРМ, к которому прикреплено оборудование (если это доп. оборудование, прикрепленное к АРМ, например телефон/сканер/принтер/игровой руль/VR шлем и т.п.)

Ниже в подсказке предлагаются АРМы, которые косвенно связаны с этим оборудованием через документы. Т.е. если по одной накладной получен комп, принтер, телефон и т.д., и комп уже введен, то при вводе телефона и принтера будет предложено привязать их к этому компу.

Если оборудование входит в состав АРМ, то все поля привязки пользователей, помещения, подразделения скрываются, значения берутся из самого АРМ.
Также скрывается поле "[Установлено в](installed_id.md)".
