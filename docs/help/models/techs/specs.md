# Оборудование → Тех. спецификация

> Отображается только в случае, если у выбранной [модели оборудования](model_id.md) стоит галочка **«индив. спеки»**

Если отобразилась, значит модель оборудования не может исчерпывающе описать, что за железо куплено в этот АРМ. В таком случае нужно плавающую часть заполнить.
Чтобы было понятнее, что нужно заполнять, рядом отображаются

- **Подсказка по заполнению спецификации** — [шаблон описания](../tech-types/comment.md) для всех моделей этого типа оборудования: что важно указать для ноутбука, моноблока, ПК или сервера.
- **Описание модели** — уже заполненная общая часть описания модели: характеристики, одинаковые у всех экземпляров (например, формат бумаги принтера, CPU роутера). Дублировать их в спецификации не нужно.
