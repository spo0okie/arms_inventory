# Сервисы/услуги → Тип объекта

Два независимых признака дают четыре комбинации. **Тип объекта**
отвечает на вопрос «что это»: *сервис* — техническая система, которая
работает на серверах/оборудовании; *услуга* — её кто-то оказывает
(человек или контрагент). **Предоставляется пользователям** отвечает
на вопрос «кто заметит, если это перестанет работать»: конечные
пользователи или только ИТ-отдел.

Примеры всех четырёх комбинаций:

| Пример | Тип | Пользователям? |
|---|---|---|
| Предоставление интернета на площадку | услуга (оказывает провайдер, «крутится» на маршрутизаторе) | да — упадёт, заметят все |
| Обслуживание кондиционеров в серверной | услуга (контрагент, есть ответственные) | нет — потребитель ИТ-отдел |
| Прокси-сервер | сервис (работает на серверах) | да — встанет, «взвоют» все |
| Синхронизация пользователей с AD | сервис (работает на серверах) | нет — проблемы заметит только ИТ |

От комбинации зависит значок в списке сервисов и подпись в подсказке
значка.
