# Маркеры

Справочник цветовых маркеров — единая палитра раскраски объектов других
справочников: [состояний оборудования](tech-states.md),
[сегментов](segments.md), L2-доменов, [категорий оборудования](tech-types.md),
состояний документов. Один маркер можно назначить нескольким объектам —
цветовая семантика остаётся общей: «опасный» красный везде один и тот же.

Маркер задаёт до трёх цветов, обязателен один:

- **Цвет фона** — главный канал: его показывают все контексты (списки,
  тёмная шапка, тултипы, карточки, ячейки карты [IPAM](../guides/ipam.md),
  компактный «уголок» в карточках оборудования). Смысл маркера должен
  читаться по одному фону.
- **Цвет рамки** и **стиль рамки** (сплошная/пунктирная) — необязательное
  уточнение (например, пунктир для DMZ-сегментов). Там, где рамку негде
  рисовать («уголок», полосы смешения в IPAM), она не показывается.
- **Цвет текста** — обычно не заполняется: подбирается автоматически
  (чёрный/белый) по яркости фона. Заполняйте только для особых случаев
  (например, тёмный фон с оранжевым текстом).

## Список

Доступен администраторам через меню «шестерёнка» → Маркеры. Названия
показываются сразу в своих цветах. Маркеры с пояснением «Базовая палитра»
созданы автоматически при переходе с прежней CSS-раскраски — их можно
переименовывать и перекрашивать как свои.

## Просмотр

Кроме полей маркера страница показывает **примеры отображения** — образец
текста на типовых фонах приложения (общий фон, тёмная шапка, тултип,
тёмная карточка, компактный уголок), чтобы оценить читаемость до
назначения маркера объектам.

## Удаление

Маркер, назначенный хотя бы одному объекту, удалить нельзя — сначала
снимите его с объектов (или назначьте им другой).
