# Контрагенты

Список организаций, с которыми приходится взаимодействовать (включая свою).

### Список

Доступен через меню "Контрагенты -> Контрагенты"

### Добавление

В списке жмем "Новый" и заполняем форму — назначение полей описано в подсказках к ним.
