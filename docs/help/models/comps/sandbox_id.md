# Операционные системы → Песочница

Песочница — изолированное окружение для ОС-двойников. Основной сценарий:
клоны и копии машин (например, ВМ, восстановленная из архива для
разбирательств, или тестовая копия продуктива) имеют **тот же FQDN**,
что и оригинал, а FQDN в учёте должен быть уникальным. Помещение копии
в песочницу решает конфликт: уникальность имени проверяется в пределах
песочницы.

Что даёт песочница:

- **уникальность имён** — «В этом домене этого окружения уже есть такой
  hostname» проверяется отдельно для каждой песочницы;
- **пометка изолированности** — если у песочницы снят признак «сетевая
  доступность», на карточке ОС не показываются ссылки удалённого
  подключения (машина недоступна из основной сети);
- **суффикс в списках** — имя песочницы выводится рядом с именем ОС,
  чтобы двойники не путались с боевыми машинами.

Пустое поле = обычная ОС в основном окружении.
