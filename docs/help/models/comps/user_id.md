# Операционные системы → Пользователь

Это поле **только для серверов и ВМ** — когда пользователь ОС отличается от
пользователя [АРМ](../techs.md). Обычно пользователь указывается у АРМ, но
бывает, что на сервере крутится виртуальная машина, одно из назначений
которой — рабочее место конкретного пользователя, при этом сам сервер не
является его личным рабочим местом. Тогда пользователь вписывается сюда, и
это отображается как сервис на ВМ.
