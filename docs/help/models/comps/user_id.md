# Операционные системы → Пользователь

Это поле **только для серверов и ВМ** (когда пользователь ОС отличается от
пользователя АРМ). Раньше пользователь был только у [АРМ](../techs.md), но
если комп — это сервер, и на нем крутится ВМ, единственное назначение (ну или
одно из назначений) которой — рабочее место пользователя, но при этом
остальной сервер не является его личным рабочим местом? В таком случае
пользователь вписывается сюда. В таком случае это отображается как сервис
на ВМ.
