# Операционные системы → Пользователь

Обычно у ОС пользователь **не указывается** — он определяется по роли
машины:

- **Рабочее место**: пользователь закрепляется за АРМ (железом), а все
  установленные на нём ОС относятся к тому же пользователю. На один АРМ
  может быть установлено несколько ОС (например, 32-битная Win7 под одно
  ПО и 64-битная Win10 под другое) — пользователя у каждой ОС дублировать
  не нужно.
- **Сервер**: пользователя у ОС нет — ею пользуются потребители сервисов,
  которые на ней работают. К серверу привязываются сервисы, а у сервисов
  есть ответственный (с ним и общаться по вопросам этой ОС).

Поле нужно для третьего случая — **виртуальная машина-рабочее место**:
сотруднику нужно дополнительное рабочее место (например, на другой
Windows или Linux), и такая ВМ крутится на сервере. К серверному
железу пользователя не привязывают, поэтому пользователь указывается
прямо в ОС.
