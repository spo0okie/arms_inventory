# Операционные системы → АРМ

Здесь руками выбирается [АРМ](../techs.md) (физический комп), на котором
установлена эта операционка. Сопоставление операционок с железом делается
вручную везде, кроме скриптов PowerCLI, которые могут сами раскидать
виртуальные ОС по гипервизорам.

По идее значение всегда должно быть установлено! Валидация пропускает пустое
значение просто потому, что скрипты добавляют ОС, не зная какой у нее АРМ,
и не могут его сразу поставить сами.

> Если к АРМ уже прикреплен пользователь, то искать АРМ можно не только
> по номеру, но и по имени пользователя
