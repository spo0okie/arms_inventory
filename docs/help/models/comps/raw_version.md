# Версия скрипта инвентаризации

Показывается в шапке ОС (в скобках рядом с датой обновления, если раскрыть
ярлычок давности данных) и отдельной колонкой в списке ОС. По суффиксу
версии видно, каким скриптом собраны данные:

- суффикс **nix** — данные собраны [скриптами для Linux/Unix](https://github.com/spo0okie/puppet_inventory_scripts);
- суффикс **powercli** — данные собраны [скриптом для VMware PowerCLI](https://github.com/spo0okie/powercli_inventory_scripts);
- пусто — ОС заведена вручную (в отдельных случаях это нормально);
- в остальных случаях записана версия [скрипта для Windows](https://github.com/spo0okie/win-inventory).
