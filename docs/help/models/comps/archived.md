# Операционные системы → Архивирован

Если ОС выведена из работы, правильнее не удалять её, а поставить в
редактировании галочку "Архивирован": она будет отмечена как не в работе
и не будет отсвечивать в общем списке (архивные записи скрываются
переключателем показа архивных).

Удалять стоит только когда история об ОС абсолютно точно не нужна
(например, временно развернутая на пару дней виртуалка) или это ошибочный
дубликат — см. раздел «Удаление» на [странице сущности](../comps.md).
