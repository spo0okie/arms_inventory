# Операционные системы → Предоставляется услугой

Поле для ОС/ВМ, которые работают **не на нашем железе**: если машина
запущена на облачной платформе виртуализации или в датацентре
провайдера, указать [АРМ](arm_id.md) невозможно — вместо него
указывается услуга, которой предоставляются вычислительные мощности.

Поля «АРМ» и «Предоставляется услугой» взаимоисключающие: ОС не может
одновременно работать на нашем оборудовании и предоставляться услугой —
форма не даст сохранить оба.

Практический смысл привязки: у облачной ВМ остаётся «адрес» в учёте —
через услугу видно, у какого провайдера и по какому договору живёт
машина, и наоборот: на карточке услуги видны все размещённые на ней ОС.
