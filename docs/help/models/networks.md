# Сети

Позволяют вести учет сетей, Vlan, IP адресов, диапазонов.

## Список

Таблица объявленных сетей: адрес/маска, VLAN, сегмент, использование
адресов. Кроме списка есть карта адресного пространства —
[IPAM](#просмотр) (меню Сети → IPAM): цветные блоки — объявленные сети
(цвет задаёт [сегмент](segments.md) — для раскраски у сегмента должен
быть заполнен код CSS), серые — свободные диапазоны, клик по свободному
блоку сразу открывает форму создания сети с заполненным адресом.

## Добавление

Форма одна и для добавления, и для редактирования — см.
[Редактирование](#редактирование). Быстрый способ — клик по свободному
блоку на карте IPAM.

## Редактирование

Форма одна и для добавления, и для редактирования. Все поля подписаны — при наведении на поле появляется подсказка.

> Если настроена опциональная [интеграция с DokuWiki](../admin/integrations/dokuwiki.md), развернутые описания сетей удобно вести в Wiki: там шире возможности редактирования и есть ссылки на другие статьи. Пишем статью в Wiki и вставляем ссылку на нее в поле **Ссылки**. Без Wiki подробное описание можно вести в поле **Подробно** (Markdown)

## Просмотр

В окне просмотра, кроме информации, внесенной непосредственно в форму редактирования, выводится также

- калькулятор сети
- адреса сети (можно переключать отображение не занятых адресов)
- входящие соединения к сервисам в этой сети (вкладка «Вх. соединения»)
- описание сегмента (если включено отображение описания сегмента в настройках)
- ссылки на привязанную вики подгружаются в отдельные вкладки на страничке (при настроенной [интеграции с DokuWiki](../admin/integrations/dokuwiki.md))

> - Если описание сети достаточно большое (по умолч. 20 строк, регулируется в настройках), оно тоже будет убрано в отдельную вкладку
> - Если описание отсутствует, то вместо него будет использоваться описание сегмента инфраструктуры
> - Также во вкладках будут подгружены ссылки на Wiki (предполагается, что там будет описание сети).
> - Если ссылок на Wiki нет, то ссылки на Wiki из сегмента инфраструктуры (предполагается, что там будет описание сегмента)

## Удаление

> ⚠ Удаление сети не блокируется, даже если внутри нее есть привязанные
> IP: адреса при этом не удаляются, а отвязываются. Принадлежность
> адреса сети вычисляется по самому адресу автоматически и
> восстановится, если сеть завести заново.
