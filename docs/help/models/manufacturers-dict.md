# Синонимы производителей

Словарь вариантов написания [производителей](manufacturers.md). Каждая запись —
одна строка, встречающаяся в отпечатках ПО/железа («hewlett-packard», «hp inc.»),
и ссылка на канонического производителя, которому она соответствует.

Используется автоматикой: при разборе отпечатков компьютеров (список железа,
список ПО) строка производителя ищется в словаре без учёта регистра
(варианты хранятся в нижнем регистре) и заменяется ссылкой на каноническую
запись. Если написание в словаре не найдено — сопоставление не происходит,
и такой вариант нужно добавить сюда вручную, привязав к нужному производителю.

Словарь пополняется по мере появления новых написаний в выгрузках.
