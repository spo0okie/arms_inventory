# Документы → Комментарий

Для счетов в комментарии желательно вести историю и логистику закупки.
Записи проставляйте в реверсивной хронологии (свежие сверху), т.к.
**первая строка комментария выводится в списке рядом со статусом
документа** — так текущий этап закупки/логистики виден сразу, без
открытия документа:

```
2020-02-12 получено на участке в Усинске, получил Сидоров С.С.
2020-02-02 отправлено на участок в Усинск, отправлял Петров П.П.
2020-02-01 получено на склад в Москве, получал Иванов И.И.
2020-01-20 оборудование отправлено, трек 12345678
2020-01-17 оплачен
2020-01-13 согласован
2020-01-10 подан на согласование
```

Если пояснение статуса не нужно — оставьте первую строку комментария
пустой.

Также в комментарии можно указывать дополнительную информацию для
облегчения поиска документа: поиск по имени документа включает в себя
и поиск по комментариям.
