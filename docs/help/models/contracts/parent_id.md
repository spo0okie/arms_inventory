# Документы → Основной документ

Здесь указывается родительский документ. Например договор для приложения к договору, счет для накладной и т.п.
Или же здесь может указываться предшественник. В таком случае надо выставить галочку **"заменяет основной документ"**.

Например, доп. соглашения к договору услуг связи: сначала действует
доп. соглашение 1, затем его замещает доп. соглашение 2 с другой датой.
У доп. соглашения 1 дата окончания может быть не указана — подразумевается,
что оно действует до даты начала замещающего его доп. соглашения 2.

При просмотре документа такие связи будут отображены в виде дерева или
цепочки; замещённые документы показываются серым с зачёркиванием.

> Техническая деталь: в базе все заменяющие версии привязываются
> к **первой** версии документа («звездой» вокруг корня) — даже если
> в форме предшественником выбрана промежуточная версия, при сохранении
> родителем станет корневая. Это нормально: в интерфейсе версии всё
> равно выстраиваются цепочкой замещения по датам.
