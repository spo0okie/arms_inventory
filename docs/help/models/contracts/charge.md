# Документы → в т.ч. НДС

Заполняется для счетов.

- Если в сумму входит 20% НДС — нажмите кнопку «20%», и величина НДС посчитается и заполнится автоматически.
- Если в счёте несколько позиций и не все содержат НДС, ровно 20% не получится — перепишите величину НДС из счёта вручную.
- Если НДС в сумме нет вообще, поле можно оставить пустым (кнопка «нет» очищает его).
