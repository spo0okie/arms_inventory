# Документы → Сумма

Если по документу предполагается оплата какой-то суммы — её надо вписать
сюда. Сумма вписывается **с учётом НДС** (величина НДС — в отдельном поле
[в т.ч. НДС](charge.md)).

Не нужно вписывать сумму в документы, которые сами по себе оплаты не
предполагают. Вместо этого к ним привязываются платёжные документы
(см. [Основной документ](parent_id.md)):

- к договору прикрепляются счета;
- к счетам — закрывающие документы (акты, УПД, накладные).

Сумму при этом несут только счета: так по цепочке документов видно,
сколько и когда фактически платили, а суммы не задваиваются при подсчётах
(например, в стоимости и долге по сервису).
