# Производители

Справочник канонических производителей оборудования и разработчиков ПО.
Каждая запись — «эталонное» написание производителя, к которому привязываются
[модели оборудования](tech-models.md) и программные продукты.

Один и тот же производитель в отпечатках железа и ПО встречается в десятках
написаний («Hewlett-Packard», «HP Inc.», «hp»...). Чтобы всё это сводилось к
одной записи, к производителю привязывается набор
[вариантов написания](manufacturers-dict.md) — при автоматическом разборе
отпечатков строка производителя ищется по этому словарю и сопоставляется
с канонической записью.

Кроме сопоставления, справочник помогает чистить названия продуктов: если
название модели/ПО начинается с имени производителя (в любом из известных
написаний), это имя отрезается, чтобы не дублировать его в списках
(«Microsoft Office» от Microsoft → «Office»).
