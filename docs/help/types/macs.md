# MAC-адреса

Списки MAC-адресов сетевых интерфейсов — атрибуты `mac` у операционных
систем и оборудования. Базовые правила ввода и поиска показываются
подсказками прямо у поля; здесь — подробности устройства.

## Хранение и нормализация

- При сохранении каждый адрес нормализуется до 12 шестнадцатеричных
  символов: `00:11:22:33:44:55`, `00-11-22-33-44-55` и `001122334455` —
  сохранится в один и тот же адрес `001122334455`.
- Диапазон хранится как упрощенные «начало-конец» `001122334455-001122334499`;
  границы при сохранении  упорядочиваются автоматически 
  (можно ввести «с большего», система поправит).

Диапазоны удобны для оборудования, которому производитель выделяет
последовательный блок адресов (например, шасси с множеством интерфейсов).

## Особенности и ограничения

- Частичное совпадение (поиск по фрагменту адреса) работает только
  по одиночным адресам и границам диапазона.
  Попадание **внутрь диапазона** проверяется только для полного MAC-адреса: 
  фрагмент, который попал бы внутрь диапазона, не находится.
- Общий синтаксис поиска (`|` — ИЛИ, `&` — И, `!` — НЕ) применим
  к MAC-адресам как к обычным строкам.
