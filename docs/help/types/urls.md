# Ссылки (список URL)

Тип данных «список ссылок»: многострочное поле, где каждая строка задает одну ссылку с текстовым описанием. Заполняется во всех объектах системы одинаково.

## Особенности

- 1 строка - 1 ссылка
- Все слова кроме последнего - описание ссылки, последнее - URL

т.е. пишем описание ссылки и в конце добавляем сам URL

```
Какое-то описание ссылки https://link.to/somwhere
Другое описание другой ссылки http://elsewhere.to/some%20path
```

Будет выведено как

[Какое-то описание ссылки](https://link.to/somwhere)  
[Другое описание другой ссылки](http://elsewhere.to/some%20path)

Пробелы внутри самого URL нужно кодировать как `%20`, иначе часть адреса будет воспринята как описание.

## Отображение

На некоторых страницах, если ссылка ведет на Wiki страничку с которой настроена [синхронизация](../admin/integrations/dokuwiki.md), то она будет загружена тут же в отдельной вкладке.
