# Синхронизация с АД

Скрипт синхронизации пользователей Active Directory с Инвентаризацией: какие поля переносятся в каждом направлении и по каким ключам сопоставляются учетные записи.

Синхронизация выполняется внешним скриптом [AD-Inventory-sync](https://github.com/spo0okie/AD-Inventory-sync), который обменивается данными с инвентаризацией через [REST API](external.md). Актуальный состав полей и ключей определяется самим скриптом.

### Возможности

#### Сценарий Inventory --> AD

Синхронизируемые поля:

- ФИО
- Организация
- Должность
- Подразделение
- Login
- Email
- Номер телефона внутренний
- Номер телефона сотовый
- ID организации
- Табельный номер
- Признак "уволен"

#### Ключ

В случае отсутствия одного из полей ключа поиск идет по следующему ключу

1. **ИНН:** UID <-> adminDescription
2. **Табельный №:** org_id <-> employeeNumber; num <-> employeeID
3. **ФИО:** name <-> displayName
4. **Логин:** login <-> sAMAccountName

#### Сценарий AD --> Inventory

Синхронизируемые поля:

- ФИО
- Организация
- Должность
- Подразделение
- Login
- Email
- Номер телефона внутренний
- Номер телефона сотовый
- Признак "уволен"

#### Ключ

1. **UID:** UID <-> objectGUID
2. **Логин:** login <-> sAMAccountName
