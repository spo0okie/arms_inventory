# Zabbix

Встроенной интеграции с Zabbix в продукте нет; при необходимости мониторинг связывается с инвентаризацией внешними средствами через REST API.

В коде приложения нет специального функционала для Zabbix. Если требуется наполнять мониторинг данными из инвентаризации (или наоборот), это делается внешними скриптами через [REST API](integrations/external.md) - по аналогии с другими [интеграциями](integrations/external.md).
