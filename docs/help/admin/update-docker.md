# Обновление (Docker)

Обновление docker-инсталляции сводится к обновлению структуры папок/конфигов из git и скачиванию свежего образа контейнера.

Для чистой [установки](install-docker.md) есть отдельная инструкция.

В папке инсталляции (с файлом `docker-compose.yml` из репозитория [arms-docker](https://github.com/spo0okie/arms-docker)) выполняем:

```bash
#обновляем структуру папок и конфигов
git pull
#скачиваем свежий образ приложения (spo0okie/inventory:v1)
docker compose pull
#пересоздаем контейнеры на новом образе
docker compose up -d
```

Миграции БД применяются автоматически при старте контейнера приложения — отдельно запускать их не нужно.

Перед обновлением не лишним будет сделать [резервную копию](backup.md).
